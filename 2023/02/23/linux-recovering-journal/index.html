# 描述

完整訊息如下，出現 recovering journal 後，要等個幾分鐘才會再出現後續的訊息。

# 修正

使用 root 權限編輯 /etc/fstab ，把 /dev/sda2 (錯誤訊息中顯示的) 那行刪除，儲存，reboot。

# 原因

應該是我前一次關機前，插外接硬碟又用 partition 管理工具 mount 外接硬碟 partition 導致。
