bind source path does not exist

# 問題描述

DevContainer 啟動 container 錯誤，查看錯誤訊息如下
Error response from daemon: invalid mount config for type "bind": bind source path does not exist: /run/user/1000/wayland-0

# 環境

  • OS: EndeavourOS
  • 在 VSCode 使用 Dev Container 套件啟動 container

# 修正方法

刪除下面兩個檔案

  • /run/user/1000/wayland-0
  • /run/user/1000/wayland-0.lock
