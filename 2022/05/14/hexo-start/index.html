# Hexo

Hexo 是基於 Node.js 的 blog 框架，使用 Markdown 或其他標記語言解析文章，產生網頁靜態檔案。

# 環境需求

安裝之前確認開發環境 Node.js 版本，Node.js 版本不可低於 8.10，官方建議使用 Node.js 10.0 或更高的版本。

# 安裝 Hexo

使用 npm 安裝 Hexo

# 檢視
